# The Wordly Dictionary App
## Description
This is a  Single Page Application (SPA) that enables users to search for a word and retrieve information such as pronunciation, definitions, synonyms, and antonyms.

## Features
- A form to submit the word you want to search
- Fetch data from an external API
- Display the details of the word
- Error handling for invalid input or unavailable words

## How to run the project
1. Fork and clone and repository
2. Open the project folder
3. Open the index.html in your browser

## Technologies Used
- HTML
- Javascript
- CSS
- Fetch API  (Free Dictionary API)
